# Как принять участие в разработке?

Если вы хотите принять участие в разработке поректа и в будющем получить особый значок в профиле мессенджера, вы должны следовать следующим правилам.

1) Создайте форк этого репозитория (нажмите [здесь](https://github.com/pulsar-chat/pulsar-web/fork))
2) Склонируйте свой форк себе:
```bash    
git clone https://github.com/your_username/pulsar-web.git
```
3) Создайте ветку для ваших изменений:
```bash
git checkout -b your-branch-name
```
4) Внесите желаемые изменения и примените их:
```
git add -A
git commit -m "your-commit-message"
git push origin your-branch-name
```
5) Откройте Pull Request
6) Ожидайте принятия и по необзодимости вносите правки
